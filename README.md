Additional information:

-It can be helpful to constrain parallel lines/points wherever possible

-Export design as a step file from FreeCad (select part- file- export- select step)

-In PrusaSlicer, import design (file- import- Import STL/3MF/STEP/OBJ/AMF...- select design)

-If using ERAS printer, check what filament is being used and config 'filament' accordingly. Select 'printer' Prusa CORE One HF0.6 nozzle

-Under 'print settings' select 0.32mm SPEED

-Print object on the flat edge (must rotate to flat edge) to avoid overhang errors
